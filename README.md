# Toronto Library Passport

Quick prototyping with TypeScript and shadcn-ui

## to run

### in terminal

`npm i`

`npm i -g json-server`

* **you may need to reload your terminal**

`json-server --watch ./_data/db.json --port 4000`

* **or the port of your choice**

`npm run dev`

go to <http://localhost:3000>
